# Mosquito infection-mediated trait variation alters temperature-dependent transmission of West Nile virus

This repository contains the data and code used in the paper:

Fay, RL et al. *Mosquito infection-mediated trait variation alters temperature-dependent transmission of West Nile virus.*

## Repository structure

- `data/` — input data used in the analyses.
- `code/` — R scripts used for data processing, model fitting, and figure generation.
- `shocket_fits` - posterior parameter draws from TPC models from Shocket et al., used in the relative R0 analyses for traits not measured in this work.
